## Install Packages
#### For frontend
```
cd frontend
npm install axios react-icons 
```
#### For backend
```
cd backend
npm install express mongoose cors nodemon
```

## Run
### Frontend
```
cd todolist_mern
cd frontend
npm run dev
```

### Backend
```
cd todolist_mern
cd backend
nodemon index.js
```
